# Firewall Automation Tool (v1.0)
## Overview
This single-file solution automates the creation of firewall rules to prevent configuration drift and manual errors.

### Prerequisites
* **Python 3.x** installed.
* **Access Keys/Credentials** for your specific provider (passed via environment variables).

---

## 🛠️ Usage Instructions
1. Save this entire block as `firewall_tool.py`.
2. Edit the `RULES_CONFIG` list inside the code.
3. Run the tool:
   ```bash
   python3 firewall_tool.py
