# Firewall Automation Tool (FireAuto)

This repository contains a single-file automation solution for managing cloud firewall rules. It uses **Terraform** to ensure that your security posture is defined as code, allowing for version control and rapid deployment.

## 🛠 The Code (main.tf)

```hcl
# Provider: Google Cloud Platform (GCP)
provider "google" {
  project = "your-project-id"
  region  = "us-central1"
}

# Resource: Automated Firewall Ingress Rule
resource "google_compute_firewall" "web_traffic" {
  name    = "auto-rule-allow-http-https"
  network = "default"

  description = "Managed by FireAuto: Allows web traffic to tagged instances."

  allow {
    protocol = "tcp"
    ports    = ["80", "443"]
  }

  # Only applies to instances with these network tags
  target_tags   = ["web-server"]
  source_ranges = ["0.0.0.0/0"]
  priority      = 1000
}
