## 🚀 The Code (main.tf)
This single block of HCL (HashiCorp Configuration Language) defines a security group rule for a cloud-based firewall (AWS/Azure style).

```hcl
# PROVIDER BLOCK: Defines where the code runs
provider "aws" {
  region = "us-east-1"
}

# RESOURCE BLOCK: The single code to create a firewall rule
resource "aws_security_group_rule" "allow_https" {
  type              = "ingress"
  from_port         = 443
  to_port           = 443
  protocol          = "tcp"
  cidr_blocks       = ["0.0.0.0/0"] # Allows traffic from anywhere
  security_group_id = "sg-12345678" # Replace with your Firewall ID
  description       = "Automated Rule: Allow HTTPS"
}
